# üá¨üáß Matrix Analysis Toolkit (English)

## üéØ Features

A comprehensive toolkit for building and analyzing the special L matrix with block structure:

- üèóÔ∏è **L Matrix Construction**: Two methods with block structure (6 row blocks √ó 4 column blocks)
  - `build_L_matrix` ‚Äî standard method
  - `generate_base` ‚Äî method using `Ew1r` for automatic nullification of linearly dependent rows
- üìä **Visualization**: Heatmaps with block boundaries, custom colormaps
- üî¢ **Eigenvalue Analysis**: Computing and visualizing the spectrum of L¬∑L^T
- üìê **Rank and Basis**: Determining matrix rank and basis rows via RREF and `generate_base`
- üßÆ **Linear Dependence**: Automatic detection and nullification of linearly dependent rows
- üíæ **Results Export**: Saving matrices and reports to files
